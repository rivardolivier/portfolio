BillETS

L’application web billETS est une plateforme qui permet de faire l’achat de billets de spectacles par Internet. Ce projet a été développé dans le cours de Technologies de développement Internet à l’ÉTS. L’application permet de consulter les spectacles offerts, réserver des billets et faire le paiements de ceux-ci.

BillETS fonctionne à l’aide de Java EE avec le framework Spring MVC. La mise en page est fait avec Bootstrap et la persistance des données se fait à l’aide du framework Hibernate connecté à une base de données postgreSQL. La base de données ainsi que le site sont hébergés sur la plateforme de développement Heroku.
